# hackernews

Node v8.11.0

Setup:
      
      Clone this repo.
      Installing dependencies -- npm install

Run:

      node hackernews --post 44
      
Tests:
      
      npm test

Modules Used:

      Axios- Promise based HTTP client for API calls
      Cheerio - Jquery like library to scrap content

Testing Modules:

      Mocha - JS testing framework
      Assert-  Assertion Engine
